# Custom Object Snap

<br>

## COS (Command)
The Custom Object Snap Add-In is a powerful AutoCAD productivity tool designed to allow users to save and quickly recall different combinations of Object Snap (OSNAP) modes.

* Save up to 9 different combinations of object snap settings in dedicated memory slots.
* Instantly recall saved snaps via commands COS1 through COS9 directly from the command line.
* Easily visualize, select all, or clear all snap modes through a user-friendly dialog.
* Configurations are saved persistently across your AutoCAD sessions.

## Download from Autodesk App Store
For the simplest experience, download the Custom Object Snap add-in directly from the Autodesk App Store. The installer handles all configuration automatically.

1. Download and install the add-in from the Autodesk Web Store.
2. At the command prompt, type COS to open the settings dialog.

## Download from GitHub
If you prefer to manage your plugins manually or are using the version from GitHub, use the Autoloader method. This ensures the add-in is loaded automatically every time you open AutoCAD.

1. Navigate to the Releases tab on the right side of this page and download the CustomObjectSnap.bundle folder.
2. Copy the folder to: %APPDATA%\Autodesk\ApplicationPlugins\
3. Launch or restart AutoCAD.
4. Approve the security prompt by clicking “Always Load.”
5. At the command prompt, enter COS.

## Workflow
The application is designed around a simple configuration and quick-recall workflow.

### Step 1: Configure Snaps
Type COS to open the Custom Object Snap settings dialog. Select your desired combination of object snaps using the checkboxes.

### Step 2: Save to a Memory Slot
Click the Save (floppy disk) icon next to one of the 9 memory slots to store the active combination. The slot will update to display the saved OSMODE value.

### Step 3: Quick Recall
Apply the settings immediately by clicking the recall button in the dialog, or apply them anytime directly from the command line using the quick commands COS1 through COS9.

## Key Features & Commands
The tool allows comprehensive control over all 14 standard AutoCAD object snaps.

### Supported Snap Modes
Endpoint, Midpoint, Center, Node, Quadrant, and Intersection.
Insertion, Perpendicular, Tangent, Nearest, and Geometric Center.
Apparent Intersection, Extension, and Parallel.
### Available Commands
COS: Opens the main Custom Object Snap settings dialog.
COS1 - COS9: Instantly applies the OSNAP configuration saved in the respective memory slot.

## About this Application
Our goal is to provide the Autodesk community with practical, high-performance tools that eliminate repetitive tasks. COS was developed to help users manage multiple OSNAP settings without having to constantly toggle individual snaps or remember specific OSMODE values.

The current version provides 9 memory slots and quick command line access for rapid configuration switching. Based on user feedback, future updates may bring additional features and improvements.
